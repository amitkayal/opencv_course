Blemishes Removal

executable name is :blemishRemove
Blemishes Removal application takes Input File as mandatory.  Size and Output file names are optional.  Size is blemish size, by default 30 (a radius 13) is considered.  When ever user presses 's' key then the image written in the output file name.

To get parameters for the user can type the following:
.\blemishRemove -h

Following is the command line parameters:

.\blemishRemove -i inputFileName -size blemishSize -o outputFileName

Following is the sample:
.\blemishRemove.exe -i ..\..\blemish.png -o ..\..\blemish_out.png

Tested on Windows Platform.
