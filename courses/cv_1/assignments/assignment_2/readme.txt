Image Crop:

executable name is :image_crop
Image Crop takes Input File and Output File as mandatory and type is optional, by default it is considered as color image.

To get parameters for the user can type the following:
.\image_crop -h

Following is the command line parameters:

.\image_crop -i inputFileName -type imageType -o outputFileName

Following is the sample:
.\image_crop.exe -i ..\..\sample.jpg -o ..\..\crop.jpg

Tested on Windows Platform.
